- ## 慢的原因
- ### 执行时间长 等待时间长
    - 查询语句
    - 索引失效
        - 创建索引
            - 单值索引 create index idx_user_name on user(name)
            - 多值索引 create index idx_user_nameEmail on user(name, email)
    - 关联查询太多join(设计缺陷或不得已的需求)
    - 服务器调优及各个参数设置
